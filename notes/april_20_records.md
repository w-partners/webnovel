# 4월 20일 - 기억의 기록

### 무의식적 충동
```
"한별아..."

마지막 만남.
호텔 방.

둘 다 알고 있었다.
내일이면 모든 게 끝난다는 것을.

"기록하고 싶어요."

한별이 먼저 말했다.

"우리를... 이 순간을..."

동철도 같은 생각이었다.
마치 무언가가 시키는 것처럼.
```

```

한별이 웃었다.
슬픈 웃음.

"아니. 나도 같은 생각했어."

옷을 벗으며.
서로를 바라보며.
카메라가 모든 걸 담았다.

마지막 사랑.
마지막 스킨십.
마지막 교감.
```

```

기억은 안 났다.

하지만 볼 때마다 눈물이 났다.

그리고 이상한 안도감.

```

```
정말 기억이 없었다.
```
