## 감정 곡선

### 동철
```
무관심 → 호기심 → 인지 → 두려움 → 갈망 →
사랑 → 죄책감 → 결단 → 희생 → 체념 → 기다림
```

### 한별
```
설렘 → 끌림 → 혼란 → 의존 → 사랑 →
죄책감 → 갈등 → 결심 → 이별 → 그리움 → 일상
```
