# 20년 후 재회 (2036년)

## 2036년 4월 20일

### 동철 (55세)
```
2016년부터 20년이 지났다.
기적적으로 암을 이겨냈다.

하지만 모든 것을 잃었다.
아내와는 결국 이혼.
아이들과도 연락 끊김.
사업도 완전히 실패.

작은 원룸에서 혼자 살고 있었다.

매년 4월 20일.
여전히 그 영상을 봤다.
이제는 화질이 낡아 보이는 오래된 영상.

"사랑해요... 백건."

21세 한별의 목소리 (2016년 당시).
```

### 한별 (41세)
```
2015년 21세에서 2036년 41세가 되었다.

결혼했다가 이혼했다.
아이는 없었다.
왜인지 아이를 가질 수 없었다.

그리고 여전히 공허했다.
무언가 중요한 것을 잃어버린 채로.

매년 4월 20일이면
이유 없이 우울했다.
그리고 그 낡은 영상을 봤다.

자신의 21세 모습.
누군지 모를 남자와의 기록.

"이 사람이 누구였을까..."
```

```

20년째 우울증 치료 중.

그리고 그가 들어왔다.
55세의 동철.

서로 알아보지 못했다.
20년의 세월이 그들을 변화시켰다.

하지만...
```

### 첫 눈맞춤
```
동철이 고개를 들었다.
한별과 눈이 마주쳤다.

순간 시간이 멈췄다.

'뭐지 이 느낌...'

한별도 동철을 바라봤다.
낯설지만 낯설지 않은 얼굴.

가슴이 뛰기 시작했다.
이유를 알 수 없는 두근거림.
```

```
"아니요... 처음 뵙는 것 같은데..."
```

```
그 이름.
영상 속에서 자신이 부르던 그 이름.

"혹시... 절 아세요?"

동철이 숨을 멈췄다.

"한별 씨?"

"네?! 제 이름을 어떻게..."
```

### 영상의 확인
```

```
