# 전생에서 결정된 15년 비틀림

## 99번째 환생 (바로 전생)

### 시대 배경
```
1920년대 일제강점기 조선

백건: 1900년생 독립운동가
한별: 1915년생 기생

15살 차이로 태어남.
```

### 비극적 엇갈림
```
1915년 - 한별 출생
백건은 이미 15세, 독립운동 시작

1930년 - 한별 15세
백건 30세, 이미 결혼하고 독립운동 중

1935년 - 한별 20세, 백건 35세
처음 만남. 하지만 백건은 일제에 체포.
```

### 15년의 기다림
```
백건이 감옥에서 보낸 편지:

"15년을 기다려주오.
내가 나오면 그때..."

하지만 백건은 1945년 광복 직전 옥사.
한별은 1950년 한국전쟁 중 사망.

정확히 15년을 엇갈렸다.
```

## 99번째 환생에서 결정된 15년 비틀림

### 백건의 결단
```
1950년, 한국전쟁 중.
죽어가는 한별을 안고.

백건: "악마여, 내 남은 수명을 가져가라.
       그리고 그 15년을 한별에게 더해라.
       다음 생에 그녀가 먼저 15년을 살게."

악마: "흥미로군. 15년을 주겠다.
       하지만 너희는 그 15년만큼 엇갈려 살 것이다."
```

### 15년 비틀림의 저주
```
백건이 자신의 15년 수명을 한별에게 주려 했지만
악마는 이를 비틀었다.

"다음 생에:
 한별이 먼저 태어나
 너는 15년 후에 꿈을 꾸며 기다리게 될 것이다.

 정확히 15년간 꿈을 꾸고
 그 후 5년의 공백을 겪어
 20년이 되어서야 만날 것이다."
```

## 100번째 환생에서의 실현

### 15년 비틀림의 작동
```
한별: 1994년생
동철: 1980년생 (14살 차이)

하지만 전생의 비틀림 때문에:

동철이 15세가 되는 1995년부터 꿈 시작.
(한별 1살)

"전생에서 백건이 준 15년이
 이번 생에서 동철의 15년 꿈으로 나타났다."
```

### 15년의 꿈과 5년의 공백
```
1995년 - 2010년: 15년간 꿈

"흩릿한 여자의 얼굴.
 누군가를 기다리는 느낌.
 15... 15... 반복되는 숫자."

2010년 - 2015년: 5년 공백

"갑자기 꿈이 멈췄다.
 마치 준비가 끝난 것처럼."

2015년: 드디어 만남

15년 꿈 + 5년 공백 = 20년 주기
```

## 15년 엇갈림의 계산

### 전생의 공식
```
백건 출생 → 15년 → 한별 출생
한별 20세 → 첫 만남 → 즉시 이별
15년 기다림 → 죽음

총 35년의 시간 중 만남은 0.
```

### 현생의 변형
```
동철 출생(1980.7.20) → 13년 7개월 → 한별 출생(1994.2.5)

전생의 정확한 15년 차이가
현생에서는 13년 7개월로 단축.

15년 - 13.6년 = 1.4년 단축

하지만 동철의 각성이 지연:
- 1995년 7월 (15세): 꿈 시작
- 2000년 7월 (20세): 완전 각성
- 2015년 8월: 실제 만남 (35세)

결과적으로 20년 주기 완성.
```

## 전생이 현생에 미친 영향

### 15년의 한(恨)
```
전생 한별의 한이 깊었다.
15년을 기다렸지만 만나지 못한 한.

그 한이 현생에 새겨졌다.

"가슴 한구석이 항상 아팠어요."
현생 한별의 고백.
"이유를 모르겠는데 15라는 숫자만 들어도..."
```

### 동철의 무의식
```
동철이 15세부터 꾼 꿈.

항상 같은 장면:
"15년을 기다려주오"
누군가의 간절한 목소리.

전생 백건이 감옥에서 쓴 편지.
그 기억이 꿈으로 나타났다.
```

## 15에서 20으로의 전환

### 업보의 누적
```
99번째 환생: 15년 엇갈림 생성
100번째 환생: 15년 + 5년 = 20년

"100번째라서 특별했어."
"모든 업보가 누적되는 순간."

99 × 15 = 1,485
1,485 ÷ 75(평균수명) = 19.8 ≈ 20년
```

### 5년의 의미
```
5 = 해방의 수
5 = 인간 의지의 개입

전생: 일제강점기 → 광복 직전 죽음
현생: 5년 지연 → 만남 가능

"5년의 지연이 오히려 해방이었어."
```

## 2036년의 깨달음

### 기억의 회복
```
재회 후 동철이 전생을 완전히 기억.

"내가 백건이었을 때...
감옥에서 15년을 약속했어."

한별도 기억하기 시작.

"기다렸어요... 15년을...
하지만 당신은 오지 않았어요."

둘 다 울었다.
전생의 한을 현생에서 풀면서.
```

### 15년 트라우마 해소
```
"이제 15년의 저주는 끝났어."

20년을 기다려 만났으니
15년의 빚을 갚은 것.

20 - 15 = 5
"5년을 더 기다린 것이 보상이야."
```
