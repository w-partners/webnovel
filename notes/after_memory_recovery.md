# 기억 회복 이후 변화

## 1월 99시간째 각성 이후

### 동철의 변화
- 전생들의 기억 완전 회복
- 100시간의 진실 깨달음
- 백건이라는 자신의 진짜 이름 인지

### 입버릇처럼 반복
```
"한별아, 4일 안에는 꼭 만나야 해."

아침 인사처럼.
헤어질 때마다.
전화할 때마다.

"4일이야. 절대 넘기면 안 돼."

한별이 물었다.
"왜 자꾸 4일이라고 해요?"

"100시간이 4일 4시간이니까."

"그게 왜..."

동철이 한별의 손을 꽉 잡았다.

"넘으면... 우리가 끝나."
```

### 일상에서의 집착
```
[문자 메시지]
동철: 지금 몇 시간째?
한별: 48시간이요
동철: 이틀 남았네
한별: 네...
동철: 내일 봐야 해
한별: 알아요

[다음날]
동철: 72시간
한별: 알아요
동철: 오늘 저녁에 꼭
한별: 네, 꼭요

[전화]
"한별아, 우리 마지막으로 본 게 언제지?"
"어제 저녁이요."
"그럼 20시간 지났네."
"대표님..."
"80시간 남았어. 3일 안에 꼭."
```

### 한별의 이해
```
처음엔 이해하지 못했다.
하지만 점점 느끼기 시작했다.

80시간이 지나면 오는 불안감.
90시간이 되면 느껴지는 적의.
95시간에 스치는 증오의 감정.

"대표님, 저도 느껴요.
4일째 되면... 대표님이 싫어지려고 해요.
무서워요."

"그래서 4일 안에 만나야 해.
절대. 절대로."
```

### 주변의 반응
```
직원 A: "대표님 왜 저렇게 4일 4일 하시지?"
직원 B: "한별씨랑 무슨 약속이라도 있나?"

아내: "요즘 4일마다 어디 가는 것 같은데?"
동철: "일이 있어서..."

한별 친구: "왜 맨날 4일 4일 거려?"
한별: "그냥... 중요한 약속이 있어서."
```

### 달력에 표시
```
동철의 수첩:
매 100시간마다 빨간 동그라미.

한별의 폰:
4일마다 알람.
"만나야 해"

둘 다 같은 날짜에 표시.
같은 시간을 카운트다운.
```

### 절박한 부탁
```
"한별아, 만약 내가 4일을 넘기면..."

"그런 일 없을 거예요."

"만약이야. 만약 넘기면...
날 찾아와. 억지로라도."

"왜요?"

"그때는 내가 너를...
아니다. 그냥 꼭 와."

"무슨 일이 있어도 4일 안에 만날게요."

"약속해."

"약속."

동철이 한별을 꽉 안았다.

"4일. 딱 4일이야.
그 이상은 우리가 감당할 수 없어."
```

## 시간 계산의 일상화

### 만날 때마다
```
"지금 리셋이니까... 100시간 시작."

시계를 확인.
폰에 타이머 설정.

"목요일 오후 2시."
"그 전에 꼭 만나야 해."
```

### 헤어질 때마다
```
"다음은 언제 볼까?"

"이틀 후?"

"그럼 48시간... 52시간 남겠네."

"충분해요?"

"아니. 하루 뒤에 보자.
24시간 후. 그럼 76시간 남아."
```

### 계획 세울 때
```
"주말에 어디 갈까?"

"토요일?"

"아니, 금요일이 98시간째야.
금요일 저녁에 만나야 해."

"그럼 금요일로."

"꼭이야. 절대 늦으면 안 돼."
```

## 4일의 의미

### 동철에게
- 운명과 싸울 수 있는 유일한 방법
- 한별을 지킬 수 있는 시간 제한
- 전생의 비극을 막는 마지노선

### 한별에게
- 점점 이해하게 되는 절박함
- 자신도 느끼는 운명의 무게
- 동철을 잃지 않기 위한 약속

### 독자에게
- 시한부 사랑의 긴장감
- 언제나 리셋해야 하는 관계
- 운명과의 끝없는 싸움

```

저녁: "내일은 꼭 만나야 해."
자기 전: "4일 넘기면 안 돼."
```

### 한별의 적응
```
처음: "왜 이렇게 집착해요?"
한 달 후: "저도 불안해요."
두 달 후: "4일 안에 꼭 볼게요."
석 달 후: "3일째예요. 내일 봐요."
```
